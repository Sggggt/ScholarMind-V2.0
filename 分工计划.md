### 2.2 数据流图

```
用户输入研究主题
       ↓
创建任务 (Task)
       ↓
M1 文献调研 ────────────────────────────────────────┐
       ├─ 搜索论文 (Brave/Semantic Scholar)            │
       ├─ 综述生成 (LLM)                               │
       └─ 产出: m1_sources.json, m1_literature_review.md│
                                                        │
M2 研究空白识别 ───────────────────────────────────────┤
       ├─ 分析文献 (LLM)                                │
       └─ 产出: m2_gap_analysis.json                    │
                                                        │
M3 Idea 生成与打分 ────────────────────────────────────┤
       ├─ 树搜索生成 Idea                                │
       ├─ 新颖性检查 (Semantic Scholar)                 │
       ├─ 三维打分 (新颖性/可行性/有趣度)                │
       └─ 产出: m3_scored_ideas.json                     │
                                                        │
M4 代码生成 ────────────────────────────────────────────┤
       ├─ 基于 AI-Scientist 模板                         │
       ├─ 静态验证 (experiment_guard)                    │
       └─ 产出: experiment.py, m4_code_gen_info.json     │
                                                        │
M5 实验设计 ────────────────────────────────────────────┤
       ├─ 运行配置生成                                    │
       ├─ 超参空间定义                                    │
       └─ 产出: m5_experiment_plan.json                   │
                                                        │
M6 实验执行 ────────────────────────────────────────────┤
       ├─ 多种子运行                                      │
       ├─ 结果捕获 (final_info.json)                      │
       ├─ 失败重试                                        │
       └─ 产出: m6_experiment_results.json                │
                                                        │
M7 结果分析 ────────────────────────────────────────────┤
       ├─ 指标分析                                        │
       ├─ 达标判断                                        │
       ├─ 不达标? → 回退 M6 (最多3次) ──────────────────┘
       └─ 产出: m7_analysis.json
                                                         │
M8 论文写作 ─────────────────────────────────────────────┐
       ├─ Stage A: 大纲生成                              │
       ├─ Stage B: 逐节撰写                              │
       ├─ Stage C: 一致性检查                            │
       ├─ Stage D: 引用 Grounding (Semantic Scholar)      │
       ├─ Stage E: 质量审计 (2轮)                        │
       └─ 产出: paper.tex, paper.pdf                      │
                                                         │
M9 评审打分 ─────────────────────────────────────────────┘
       ├─ 多审稿人模拟
       ├─ 四维评分 (Soundness/Presentation/Contribution/Overall)
       └─ 产出: m9_review_report.json
```

## 三、后端详细分工

### 3.1 核心入口

| 文件 | 职责 | 说明 |
|------|------|------|
| `main.py` | FastAPI 应用入口 | 服务启动、CORS、静态文件托管 |
| `config.py` | 全局配置管理 | 环境变量、路径、默认值 |
| `runtime_config.py` | 运行时配置 | 动态绑定配置，支持运行时修改 |

### 3.2 API 层 (api/)

| 文件 | 职责 | 主要接口 |
|------|------|----------|
| `routes.py` | REST API 路由 | `/api/tasks`, `/api/chat` 等 |
| `schemas.py` | 请求/响应模型 | Pydantic 数据模型 |
| `ws.py` | WebSocket 管理 | 实时推送 (计划中) |

### 3.3 流水线层 (pipeline/)

| 文件 | 职责 | 核心功能 |
|------|------|----------|
| `orchestrator.py` | 流水线编排器 | 模块调度、上下文传递、错误处理 |
| `state.py` | 任务状态机 | 暂停/恢复/终止、审阅等待 |
| `tracer.py` | 日志追踪器 | 数据库日志、WebSocket 广播 |

### 3.4 研究模块层 (modules/)

| 模块 | 文件 | 输入 | 输出 | 技术方案 |
|------|------|------|------|----------|
| M1 | `m1_literature.py` | 研究主题 | 论文列表、综述 | GPT-Researcher + 搜索 API |
| M2 | `m2_gap_analysis.py` | 文献综述 | 研究空白列表 | LLM 分析 |
| M3 | `m3_idea_scoring.py` | M1+M2 结果 | 带评分 Idea | 树搜索 + 新颖性检查 |
| M4 | `m4_code_gen.py` | 选中 Idea | 实验代码 | AI-Scientist 模板 |
| M5 | `m5_experiment_design.py` | 实验代码 | 实验计划 | LLM 规划 |
| M6 | `m6_agent_runner.py` | 实验计划 | 实验结果 | Subprocess + 重试 |
| M7 | `m7_analysis.py` | 实验结果 | 分析报告 | LLM 分析 + 回退判断 |
| M8 | `m8_paper_writing.py` | 全部结果 | LaTeX + PDF | 5 阶段写作 + pdflatex |
| M9 | `m9_review.py` | 论文 | 评审报告 | 多审稿人模拟 |

### 3.5 基础设施 (modules/)

| 文件 | 职责 | 功能 |
|------|------|------|
| `base.py` | 模块基类 | 统一模块接口 |
| `llm_client.py` | LLM 客户端 | 统一 LLM 调用、重试机制 |
| `ai_scientist_bridge.py` | AI-Scientist 适配 | 论文搜索、实验模板 |
| `experiment_guard.py` | 实验代码验证 | 静态检查、关键词匹配 |
| `experiment_sim.py` | 实验结果模拟 | 降级数据生成 |
| `ssh_runner.py` | SSH 远程执行 | 分布式实验 (计划) |

### 3.6 服务层 (services/)

| 文件 | 职责 | 功能 |
|------|------|------|
| `task_service.py` | 任务服务 | 创建/运行/恢复/终止任务 |
| `conversation_service.py` | 会话服务 | 对话管理、消息存储 |

### 3.7 数据层 (db/)

| 文件 | 职责 | 说明 |
|------|------|------|
| `database.py` | 数据库连接 | SQLite + aiosqlite |
| `models.py` | ORM 模型 | Task, TraceLog, TaskOutput, ChatSession, ChatMessage |
